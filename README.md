## 📝 Come Lavoriamo Insieme

**Cosa deve fare ognuno:**
1. Fare commit regolari con messaggi chiari (es: "Aggiunto menu di navigazione")
2. Controllare il codice degli altri (code review)
3. Aprire issue quando si trova un bug o si ha un'idea
4. Documentare quello che si fa

**Regole semplici:**
- Prima di iniziare a lavorare: `git pull` per scaricare le novità
- Dopo aver lavorato: `git add .` → `git commit -m "messaggio"` → `git push`
- Comunicare nel gruppo se ci sono problemi

---

## 📅 Scadenze e Obiettivi

- Rispettare le deadline concordate
- Aiutarsi a vicenda quando qualcuno è in difficoltà
- Comunicare sempre se ci sono ritardi o problemi
- Tenere aggiornato il gruppo sui propri progressi
